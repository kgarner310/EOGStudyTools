# Evie's EOG Study Tools 🔬

Interactive science study tools for 5th grade NC Science EOG. Built for Evie — open `index.html` or visit the live site to start studying.

## Tools

| Tool | Description |
|------|-------------|
| 🏠 Hub | XP system, level progression, domain mastery, badges |
| 📝 Practice Quiz | 44-question EOG-format test with explanations |
| 🃏 Flashcards | 60 flip cards across all 3 domains |
| ⚔️ Boss Battle | 3 bosses × 10 questions — Matter Monster, Ecosystem Overlord, Earth Shaker |
| 🥎 Batting Practice | Softball-themed game — correct answers = hits |
| ⚡ Speed Trivia | 30-second countdown, streak multiplier, 4 domain modes |
| 💧 Water Cycle | Follow Droppy the water droplet through all 8 stages |
| ⚗️ Matter Sorter | Swipe cards to sort changes, matter types, and properties |
| 🌿 Food Web Builder | Tap to connect organisms, then answer "What If?" scenarios |
| ⚡ Forces Lab | Interactive sliders — see balanced vs. unbalanced forces live |
| 🎰 Scratch Cards | Canvas scratch-off cards revealing science facts |
| 📋 EOG Tips | Test strategies, day-of checklist, domain quick reference |

## Domains Covered

- **Physical Science** — Forces, matter, physical/chemical changes, mixtures
- **Life Science** — Cells, ecosystems, food webs, traits, body systems
- **Earth Science** — Rocks, water cycle, Earth's layers, weathering

## Tech

Plain HTML/CSS/JS — no frameworks, no build step, no server needed. Progress shared via `localStorage`.
